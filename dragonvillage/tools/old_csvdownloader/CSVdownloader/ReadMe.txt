
[시작하기]

    1. CSVDownloader 폴더 내의 CSVDownloader.exe를 실행합니다.

    1-1. Java Runtime이 없다면 자동으로 설치할수 있는 웹페이지로 접속됩니다.
    웹페이지 안내에 따라 Java Runtime을 설치합니다.

    1-2. 최초 실행시에는 구글 드라이브에 관한 권한 승인 요청하는 웹페이지가 나옵니다. 권한을 승인하면 자동으로 실행됩니다.


[사용법]

    0. 앱 구동시
* 구글 드라이브 로드하는 시간이 필요합니다.
* 콘솔 출력창에서 로딩이 끝난 후에 다운로드 가능합니다.

    1.1 선택하여 다운로드

* 상단 텍스트바에 검색할 문서명을 입력합니다. 또는 아무것도 입력하지 않으면 전체 문서를 볼 수 있습니다.
* 리스트에 나온 문서 중 필요한 문서를 다중 선택하여 우측 버튼 중 최상단에 있는 download selected document 버튼을 클릭합니다.
* 문서를 저장할 경로를 선택할 수 있습니다. 즉시 팝업되는 경로는 CSVDownloader.exe 가 있는 폴더를 기준으로 합니다.
* 다운로드가 완료되면 팝업이 나옵니다.


    1.2 프리셋 다운로드
* 상단 메뉴 중 preset을 클릭하면 ./preset 폴더에 있는 텍스트 파일을 읽어와 선택가능하도록 되어 있습니다.
* preset용 텍스트 파일은 엔터(개행문자)로 구분됩니다.
* preset용 텍스트 파일으 이름이 메뉴의 항목으로 나옵니다.
* preset을 선택한 후 preset download 버튼을 누르면 해당 파일들을 다운로드 합니다.

    1.3 Refresh
* 구글 드라이브가 갱신되었을 시에는 해당 버튼을 클릭하며 문서리스트를 갱신합니다.

    2.  SVN Commit

* 우측 하단 SVN Commit 버튼을 클릭하면 선택한 경로의 폴더를 커밋할 수 있습니다.
